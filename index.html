Deployment & Hosting Guide

คุณชุมนุมสามารถนำไฟล์หน้านี้ไปเปิดใช้งานเป็นเว็บไซต์ของ **หจก.คาร์บอนโซลูชั่น (ไทยแลนด์)** ได้ฟรีและรวดเร็วที่สุดโดยทำตามขั้นตอนด้านล่างนี้ครับ

1. GitHub Pages (แนะนำ)

  •  สร้างบัญชีที่ GitHub.com (ฟรี)
  •  สร้าง Repository ใหม่ชื่อ carbon-app
  •  อัปโหลดไฟล์ index.html นี้ขึ้นไป
  •  ไปที่ Settings > Pages > เลือกเปิดใช้งาน

2. Vercel (เร็วและเสถียร)

  •  ไปที่ Vercel.com เลือก Hobby Plan (ฟรี)
  •  เชื่อมต่อกับ GitHub หรือลากไฟล์วาง
  •  ระบบจะออกลิงก์เว็บให้ทันทีภายใน 1 นาที
  •  รองรับ SSL (HTTPS) อัตโนมัติ

เชื่อมต่อโดเมน carbonsolutionthailand.com

1

เข้าสู่ระบบจัดการโดเมนของคุณชุมนุม (เช่น GoDaddy, DotArai) และไปที่เมนู DNS Management

2

เพิ่มค่า DNS Records ดังนี้ (สำหรับ Vercel):

Type: A Name: @ Value: 76.76.21.21

Type: CNAME Name: www Value: cname.vercel-dns.com

3

รอการกระจายตัวของ DNS (ประมาณ 30-60 นาที) เว็บไซต์จะเข้าใช้งานได้ผ่านชื่อโดเมนหลักทันทีครับ
